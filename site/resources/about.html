How pastebot works

Pastebot is fundamentally very simple. He is three docker containers, all of which run completely independently. One is this website, one is the API which proxies discord, and one is the bot itself. These are networked with docker-compose, each providing its specific function.
I host the bot on a Linode server in Compose, though it could be sharded or scaled very efficiently, using round-robin DNS or efficient load balancing.
